# geo-file-type
Detect the Types of Geospatial Input, including ASCII Grid, Auxiliary XML, GeoTIFF, JPG, PNG, SHP (Shapefile), and World File

# features
- Memory Safe: avoids loading the whole file into memory

# file types
| Name | Supported |
| ---- | --------- |
| ASCII Grid | Yes |
| Auxiliary XML| Yes |
| Cloud Optimized GeoTIFF | Yes |
| GeoRaster | Yes |
| GeoJSON | No |
| GeoTIFF | Yes |
| NetCDF | Yes |
| JPG | Yes |
| PNG | Yes |
| PRJ | No |
| SHP (Shapefile) | Yes |
| TopoJSON | No |
| World File | Yes |
